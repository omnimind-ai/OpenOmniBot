04

注意事项

正式接入面

外部 agent 应优先使用 MCP tools：oob_function_list、 oob_run_log_convert、oob_function_guard_check、 oob_function_run。 Skill 和 docs 负责激活条件与操作约束，不承担可信执行。

Codex CLI Sandbox

read-only 和 workspace-write 会阻断 loopback MCP HTTP。 Codex 外部触发验收使用 danger-full-access，同时 prompt 明确禁止文件修改。
